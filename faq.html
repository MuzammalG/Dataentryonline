There are a number of reasons the tasks providers will decline a tasks and deem it as unsuccessful.
  • Always answer honestly – this means we can continue to provide you with more tasks that are relevant to you.
  • Complete the task once it’s been opened – once you’ve started a task, please don’t leave it to complete later; the task will most likely ‘time out’ and will be deemed as unsuccessful
  • Only use one device – make sure you complete a task on the same device that you started it on; if you try and carry on a task on a different device, the task provider may class this as ‘suspicious activity’ as it may look like you’re trying to complete the same task more than once.
  • Use a valid internet connection – if you’re using a VPN, proxy or Tor, please make sure you turn it off before taking a task; our task providers are less likely to decline your task if you’re using a valid IP connection.
  • Don’t use offensive language – it is not tolerated by our task providers and will result in an immediate ban. Some of our providers have a zero-tolerance policy towards fraudulently completed surveys and will result in an immediate ban with no warning.
